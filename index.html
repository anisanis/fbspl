Frequently asked questions

How does it work?

FbSpl takes advantage of Facebook's support of [[ID]] (where ID can be the Numerical ID or Username of either a Profile or Page).

I made a page for each and single emoticon and wrote a userscript that transcribes common emoticon format into [[ID]] format.

I want to make my own emoticon set, can I?

Of course! FbSpl is opensource and can be forked on GitHub. All what you have to do is to create Facebook pages, update IDs and urls in the script and host them somewhere!

However, you should keep it up to date whenever a bugfix is added.

Where can I suggest new features or report bugs?

You can always post an new issue Here!
